READ ME
Steps to run project:

Program uses Python 3.7, numpy, opencv, and PriorityQueue libraries
At least 3.7 is needed as dictionary should be in order for animation portion to work

Set start and goal node, located at the top of the .py file as values 'x_start', 'y_start', 'x_goal', and 'y_goal'

Enter 'python3 Dijkstra-pathplanning-Qamar-Syed.py' into command line or run from IDE

Program will print to command line if nodes are invalid 

Animation saved as an .avi file using xvid codec in the project directory, these settings work on Ubuntu 20.04 can be changed if necessary on line 196
